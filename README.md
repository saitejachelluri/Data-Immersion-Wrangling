# Data Immersion & Wrangling

## Project Overview
This project focuses on performing data immersion, data quality assessment, data cleaning, and transformation on a retail sales dataset. The primary objective is to prepare a clean, structured, and analysis-ready dataset for further business intelligence and analytical tasks.

---

## Objectives
- Understand the raw dataset structure
- Perform data quality assessment
- Detect missing values and duplicate records
- Standardize data formats
- Perform feature engineering
- Create analytical visualizations
- Export a final cleaned dataset

---

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## Data Cleaning & Transformation
The following operations were performed during the project:

- Data Familiarization
- Missing Value Analysis
- Duplicate Record Validation
- Data Type Conversion
- Date Standardization
- Feature Engineering
- Outlier Detection
- Correlation Analysis
- Distribution Analysis
- Final Data Validation

---

## Features Engineered
New analytical columns were created from the dataset:

- Order Year
- Order Month
- Order Day

These features help improve business analysis and reporting capabilities.

---

## Visualizations Included
- Missing Values Heatmap
- Correlation Matrix
- Sales Distribution Plot
- Outlier Detection Boxplot

---

## Final Outcome
The dataset was successfully cleaned, transformed, and validated for analytical purposes. The final cleaned dataset is analysis-ready and suitable for further Exploratory Data Analysis (EDA), dashboarding, and business intelligence tasks.
